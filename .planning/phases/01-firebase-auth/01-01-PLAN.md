```bash
curl -o lib/firebase-app-compat.js "https://www.gstatic.com/firebasejs/12.12.0/firebase-app-compat.js"
curl -o lib/firebase-database-compat.js "https://www.gstatic.com/firebasejs/12.12.0/firebase-database-compat.js"

```

For fonts: Fetch the Google Fonts CSS2 API response to extract the actual woff2 static URLs, then download each woff2 file:
- Instrument Serif Regular (latin subset)
- Instrument Serif Italic (latin subset)
- Outfit Variable (latin subset, weight range 100-900)

Use curl with a browser User-Agent header to get woff2 URLs from the CSS2 API:
```bash
curl -H "User-Agent: Mozilla/5.0" "https://fonts.googleapis.com/css2?family=Instrument+Serif:ital@0;1&display=swap" 2>/dev/null | grep -o 'https://[^)]*\.woff2'
curl -H "User-Agent: Mozilla/5.0" "https://fonts.googleapis.com/css2?family=Outfit:wght@300;400;500;600;700;800&display=swap" 2>/dev/null | grep -o 'https://[^)]*\.woff2'
```

Download the latin subset woff2 files to lib/ with the names: InstrumentSerif-Regular.woff2, InstrumentSerif-Italic.woff2, Outfit-Variable.woff2.

Note: The Anton font referenced in the current Google Fonts link is NOT used anywhere in the app CSS — do not download it.

Verify all files are non-empty after download:
