<objective>
Bundle the Firebase Storage SDK, implement cloud upload of NDA artifacts (images + video) on completion, and write Firestore NDA documents with full metadata. This creates the persistent cloud record of every NDA.

Purpose: Local IndexedDB storage is device-bound and video blob URLs don't survive app restarts. Cloud storage makes NDAs accessible across devices and ensures video evidence is preserved. The Firestore NDA documents enable the "My NDAs" view, usage counting, and remote signing in later phases.

Output: On NDA completion, artifacts upload to Cloud Storage and a Firestore document is created. Existing local vault save continues to work (dual storage for offline capability).
</objective>

**Step 2:** Add helper to convert base64 data URL to Blob:
```javascript
function dataURLtoBlob(dataURL) {
  const parts = dataURL.split(',');
  const mime = parts[0].match(/:(.*?);/)[1];
  const raw = atob(parts[1]);
  const arr = new Uint8Array(raw.length);
  for (let i = 0; i < raw.length; i++) arr[i] = raw.charCodeAt(i);
  return new Blob([arr], { type: mime });
}
```

Place this in the auth functions section (after signOutUser, before components), as it's a utility used by the upload function.

**Step 3:** Add uploadNdaToCloud function in the auth functions section:
```javascript
const uploadNdaToCloud = async (ndaId, record, videoBlob) => {
  const user = firebase.auth().currentUser;
  if (!user) return;
  const basePath = `users/${user.uid}/ndas/${ndaId}`;

  // Upload NDA image
  let ndaImagePath = null;
  if (record.ndaImage) {
    const ndaBlob = dataURLtoBlob(record.ndaImage);
    const ndaRef = storage.ref(`${basePath}/nda.png`);
    await ndaRef.put(ndaBlob);
    ndaImagePath = ndaRef.fullPath;
  }

  // Upload receipt image
  let receiptImagePath = null;
  if (record.receiptImage) {
    const receiptBlob = dataURLtoBlob(record.receiptImage);
    const receiptRef = storage.ref(`${basePath}/receipt.png`);
    await receiptRef.put(receiptBlob);
    receiptImagePath = receiptRef.fullPath;
  }

  // Upload video
  let videoPath = null;
  if (videoBlob) {
    const videoRef = storage.ref(`${basePath}/video.mp4`);
    await videoRef.put(videoBlob);
    videoPath = videoRef.fullPath;
  }

  // Write Firestore NDA document
  await firestore.collection('users').doc(user.uid)
    .collection('ndas').doc(ndaId).set({
      party1: record.party1,
      party2: record.party2,
      purpose: record.purpose,
      state: record.state,
      date: record.date,
      categories: record.categories,
      type: 'nda',
      ndaImagePath: ndaImagePath,
      receiptImagePath: receiptImagePath,
      videoPath: videoPath,
      bioVerified: record.bioVerified,
      bioResult: record.bioResult,
      verification: record.verification,
      status: 'complete',
      createdAt: firebase.firestore.FieldValue.serverTimestamp()
    });
};
```

**Step 4:** Modify saveToVault to also trigger cloud upload:
```javascript
const saveToVault = useCallback(async (finalNdaImage, finalReceiptImage, finalBioResult, finalVideoURL) => {
  const ndaId = generateSessionId();
  const record = {
    id: ndaId,
    party1: form.party1, party2: form.party2,
    purpose: form.purpose, state: form.state,
    date: form.date, categories: form.categories,
    ndaImage: finalNdaImage,
    receiptImage: finalReceiptImage || null,
    videoURL: finalVideoURL || null,
    bioVerified: !!finalBioResult,
    bioResult: finalBioResult || null,
    verification: [finalVideoURL && "Video", "Signature", finalBioResult && "Biometric"].filter(Boolean).join(" + "),
    createdAt: Date.now()
  };

  // Save locally first (instant)
  await vaultSave(record);
  setVaultCount(prev => prev + 1);

  // Upload to cloud (async, don't block UI)
  try {
    await uploadNdaToCloud(ndaId, record, videoBlobRef.current);
  } catch (e) {
    console.error('Cloud upload error:', e);
  }

  // Clear video blob ref
  videoBlobRef.current = null;
}, [form]);
```

**Important:** The cloud upload happens after local save. If it fails (offline, etc.), the NDA is still saved locally. No upload progress indicator in this plan — the user sees "Saved to Vault" immediately. Cloud upload runs in background.
